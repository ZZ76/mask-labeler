# mask-labeler

### Usage
python labeler.py img</br>
--optional img indicates which file to start in the list. Otherwise start from the first one.</br>

### Shortcuts
1~5: select brush size</br>
n: next</br>
p: previous</br>
s: save</br>
esc: exit</br>

b: brush</br>
e: erase</br>
g: show/hide grids</br>
r: show/hide mask</br>
x: high contrast on green channel</br>
